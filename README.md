# Análise de Dados de Clientes de um Banco

Programa em Python para analisar dados de uma base de dados de clientes de um banco fictício

Este programa constroi gráficos que relacionam os dados dos clientes, a fim de descobrir quais os principais motivos do cancelamento de suas contas bancárias

Para executar este projeto, é necessário ter o Pyhton instalado na máquina e as seguintes bibliotecas:

- pandas
- plotly.express
- virtualvenv (opcional)
